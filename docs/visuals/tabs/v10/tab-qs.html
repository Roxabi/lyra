What is Lyra?

Lyra is a personal AI agent engine that runs as a persistent daemon on your home server. It connects to Telegram and Discord, routes incoming messages to the right AI agent, and streams responses back. Each agent is a full conversation partner: it has a system prompt, a set of tools, a model, and optional persona.

Unlike hosted chatbots, Lyra runs entirely on your own hardware — your GPU, your data, your rules. Agent configuration lives in a local SQLite database and is hot-reloaded on every message, so edits take effect instantly without a restart.

How a message flows

Telegram / Discord → Adapter → InboundBus → Hub → Pool (scope) → AgentBase → OutboundDispatcher → Telegram / Discord

Each step is async. The InboundBus uses a per-platform queue. The OutboundDispatcher has per-bot retry queues with circuit breakers. Agent execution is isolated per Pool — one slow conversation doesn't block others. Token-level streaming flows from CLI adapters back to the platform.
