Storage
~/.lyra/auth.db
SQLite database (WAL mode). Contains agents, bot_agent_map, agent_runtime_state tables. Primary source of truth for all agent config.
~/.lyra/config.toml
Runtime config: API tokens, [defaults] (cwd), logging, platform settings. Machine-specific — gitignored.
~/.lyra/agents/
User-level agent TOML overrides. Higher precedence than src/lyra/agents/. Machine-specific — gitignored.
src/lyra/agents/
Bundled system default agent TOML files. Committed to the repo. Lowest precedence.
~/.lyra/memory/
Agent memory files (namespace-scoped). Persisted across sessions.
Message Flow
User message → Platform adapter → Command router → Agent pool → Backend driver → LLM → Stream back → Reply sent
Voice message → STT (Whisper) → text transcript → normal flow → TTS (Chatterbox) → audio reply
