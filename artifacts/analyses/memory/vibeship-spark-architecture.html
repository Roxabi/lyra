Quality Gates

What Guards the Pipeline

Meta-Ralph
Multi-dimensional quality scoring: actionability, novelty, reasoning, specificity, outcome-linked. Roasts proposed learnings before storage. "Evolve, don't disable."
Memory Gate
Weighted scoring (impact 0.30, novelty 0.20, surprise 0.30, recurrence 0.20). Threshold at 0.50. High-stakes keywords get automatic pass.
Distillation Transformer
Suppression layer: catches noise that passes Meta-Ralph. Checks for code artifacts, tautologies, missing action/reasoning. Unified score floor at 0.20.
Advisory Gate
Cooldown, deduplication, authority levels (silent/whisper/note/warning/block), fallback rate limiting, budget caps per window.
Promotion Gate
Threshold 0.70 reliability, minimum 3 validations, confidence floor 0.90. Rate-limited to max once per hour. Promotes to CLAUDE.md / AGENTS.md.
