  • loadClaudeMd(cwd) — reads global + project CLAUDE.md

Semantic search with automatic query expansion. Default for content queries.

Full document retrieval by file path or doc ID.
multi_get
Glob-based batch retrieval. Used for date-range queries.
Key constraint: All hooks are synchronous (block until complete). The pgrep -f "qmd.*embed" guard prevents concurrent embedding processes — if one session's embed is running, others skip it. The next embed catches pending hashes.
Configuration: config.json persists outputDir (markdown destination) and loadContextOnStartup (whether fresh sessions get recent turns). Setup wizard in SKILL.md walks through interactively: output dir, conversion, qmd collection, embeddings, MCP server, hooks, CLAUDE.md guidance.
