Quick Start
1
Install
Clone and install dependencies with uv.
bash
git clone https://github.com/roxabi/lyra && cd lyra
uv sync
2
Configure
Copy config.toml.example → ~/.lyra/config.toml and set tokens + API keys.
3
Seed the DB
Import bundled agent TOML files into ~/.lyra/auth.db.
bash
lyra agent init       # seed TOML → SQLite
lyra agent list       # verify agents loaded
4
Assign to a bot
Link an agent to a Telegram or Discord bot ID.
bash
lyra agent assign lyra --platform telegram --bot <bot_id>
5
Start
Launch both adapters via supervisord.
bash
make lyra            # start lyra_telegram + lyra_discord
make lyra status     # verify running
