# Spec: Per-Role NKeys ACL Matrix (Issue #706)

## Scope

This spec defines the canonical publish/subscribe permissions for each NATS identity
in the Lyra ecosystem. It is the source of truth for `deploy/nats/acl-matrix.json`
and is verified by `scripts/check-acl-matrix-spec.sh` on every CI run.

**Updated (postmortem Fix 1+2, 2026-04-28):**
- Retired `tts-adapter` and `sst-adapter` (replaced by `voice-tts`/`voice-stt`)
- All inbox subjects normalized to lowercase `_inbox.X.>` (ADR-051)
- Added `image-worker` and `clipool-worker` with explicit `_inbox.hub.>` publish grants
