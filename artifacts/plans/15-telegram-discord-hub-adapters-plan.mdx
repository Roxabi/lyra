Shape-B routing model (`RoutingKey(platform, bot_id, user_id)` + typed `PlatformContext`) wired
to Telegram (aiogram v3 webhook) and Discord (discord.py v2 Gateway) adapters, running as
co-routines in a single asyncio event loop launched via `python -m lyra`. All 4 slices
implemented and green (103 tests passing).
