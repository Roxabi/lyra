## Status

Superseded by [ADR-047](047-nats-connector-ownership-pattern.mdx)

> **Note (#886):** This ADR originally described a lyra-owned standalone supervisor
> under a repo-local `deploy/` subtree. That directory has been removed. Lyra is now a
> **tenant** on the machine-level supervisord at `~/projects/supervisord.conf`, with
> conf files at `~/projects/conf.d/` (symlinked from `deploy/conf.d/` via `make
> register`). Production is Podman Quadlet (ADR-053). The systemd unit targets
> `%h/projects/scripts/start.sh --all`.
>
> The original hub-and-spoke body (Decision, Consequences, Implementation) has been
> removed to keep the repo free of stale path references. See git history pre-#886
> for the historical text, or ADR-047 / ADR-053 for the current decisions.
